#Compilation project
[Homepage](http://www.ensiie.fr/~guillaume.burel/compilation/)

##Prerequisites
You need to have `ocaml`, `ocamlyacc` and `ocamllex` installed.

##Use
To compile the project, type:

	make

And then to launch it:
	
	./pseudopascal
	
##Clean
To clean the directory, type 

	make clean
